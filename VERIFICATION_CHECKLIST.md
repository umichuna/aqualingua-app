## ✅ チェック項目

### 1. ジャンル削除バグ修正
**シーン**: 管理画面 → ジャンル管理

- [ ] カスタムジャンルを1個追加（例: テスト用）
- [ ] テスト用ジャンルで単語を1個作成
- [ ] 管理画面でテスト用ジャンルの「×」ボタンをクリック
- [ ] **確認ダイアログが出る**「『テスト用』を削除すると、このジャンルを持つ単語1件のジャンルも空白になります。よろしいですか？」
- [ ] 「削除する」ボタンをクリック
- [ ] ✅ ジャンルが消える・単語帳でそのジャンルが空白になる

**チェックポイント**:
- 確認ダイアログなしに削除できない
- 削除後、単語のジャンルが空白（""）になっている

---

### 2. 全単語に続ける機能 削除確認
**シーン**: しごと → 聞き流し

- [ ] しごト画面で「聞き流し」モードを選択
- [ ] 設定を確認
- [ ] **「全語を聞き終わった後の動作」セクションがない**（ラジオボタン2個の選択肢なし）
- [ ] 説明「聞き流し（無限）。絞り込みをリピート…」のみ表示
- [ ] 再生ボタン「🎧 聞き流す」を押す
- [ ] 単語を1周してリスト最後に到達
- [ ] ✅ 先頭に戻ってリピート（「全単語に切り替わる」動作がない）

**チェックポイント**:
- 「🔀 全単語に続ける」ラジオボタンが表示されていない
- 常に同じ絞り込みセットをリピート

---

- [ ] 単語帳画面を開く

**チェックポイント**:
- ページ再読み込みなしにジャンルが反映される

### 4. 聞き流し言語方向対応（既実装確認）
**シーン**: しごと → 聞き流し → 設定

- [ ] 聞き流しモードで「言語方向」セクションを確認
- [ ] 「🇬🇧 English → 🇯🇵 日本語」がデフォルト選択
- [ ] 「🇯🇵 日本語 → 🇬🇧 English」に切り替え
- [ ] 再生ボタンで開始
- [ ] ✅ 音声が「日本語（意味）→ 英語（綴り）」の順で流れる

**チェックポイント**:
- 言語方向の選択が聞き流し再生に反映される
- 両方向とも正しく再生されている

### 5. 自己採点UI 2ステップ化（既実装確認）
**シーン**: しごと → 自己採点
